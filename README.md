# Parachute


But du jeu:
Le joueur contrôle un rameur dans une barque, grâce aux 2 boutons (gauche et droite) il est possible de déplacer le rameur sur 3 positions. 
Un hélicoptère placé en haut à droite de l'écran largue régulièrement des parachutistes. 
Le joueur doit rattraper le parachutiste avant qu'il ne tombe à la mer et se fasse dévorer par les requins. 
Chaque parachutiste qui va à la mer donne au joueur un malus matérialisé par une tête de requin à droite de l'écran. 
Au bout de 3 malus, la partie est terminée. 


Reste à faire:
Si le joueur atteint 200 points puis 500 points (1 point par parachutiste récupéré), les malus sont supprimés et une petite musique retenti. 
Améliorer le code et le gameplay pour régler la vitesse => Vos conseils et propositions sont les bienvenus !!
Caler les bips sons sans qu'ils deviennent pénible et ajouter l'option sons actifs ou non via les options
Trouver et ajouter une musique activable ou non via les options => Vos propositions de code pour faire une musique de fond pas trop répétitive et agréable sont les bienvenues.


Différences principales par rapport à l'original:
Dans le jeu original, il existe 2 type de jeu, type A et type B. 
La différence du jeu en type B est que les parachutistes peuvent ou non rester accrochés à un palmier et se décrocher de façon aléatoire ce qui augmente la difficulté du jeu.
